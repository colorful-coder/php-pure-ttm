# php-pure-ttm

CRUD with prue PHP

## Features

* Editor

## How-to

* Intstall Xampp
* open Apache and MySQL
* add project folder in <code>c:/xampp/htdocs/</code>
* run <code>http://localhost/phpmyadmin/</code> to create DB table
* run <code>http://localhost/project_folder_name/file_name/</code>
